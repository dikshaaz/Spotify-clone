Spotify Clone 

A static Spotify Clone built using HTML and CSS.
This project focuses on recreating the Spotify user interface to practice frontend layout design, styling, and responsiveness.

 Features
- Spotify-inspired UI design
- Navigation sidebar
- Music playlist layout
- Footer music player UI (static)
- Responsive design for different screen sizes

Note: This is a UI clone only. No music playback or dynamic functionality is implemented.

Tech Stack
- HTML5
- CSS3
